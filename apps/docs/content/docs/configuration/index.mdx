## Data Storage

data-peek stores data locally on your machine:

| Data | Location | Encrypted |
|------|----------|:---------:|
| Connections | App data folder | ✓ |
| Query history | App data folder | - |
| Saved queries | App data folder | - |
| License | App data folder | ✓ |
| Settings | App data folder | - |
| Window state | App data folder | - |

<Callout type="info">
  Connection credentials (passwords) are encrypted with a device-specific key.
</Callout>

## Privacy

data-peek is privacy-first:

- **No telemetry** - No usage data collected
- **No analytics** - No tracking scripts
- **No cloud** - All data stays on your machine
- **No accounts** - No sign-up required
- **Local storage** - Credentials never leave your device
