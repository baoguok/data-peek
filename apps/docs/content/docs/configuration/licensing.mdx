# Licensing

data-peek uses an honor-system licensing model inspired by sustainable indie software.

## License Types

### Personal (Free)

Use data-peek for free if you're:
- Working on personal projects
- Learning or studying
- Contributing to open source
- A solo founder (company of one)
- A student or educator

**All features are included.**

### Pro ($29/year)

A Pro license is required for:
- Commercial use at for-profit companies (2+ people)
- Freelancers billing clients
- Agency work

**Benefits:**
- Commercial use rights
- 3 device activations
- 1 year of updates
- Perpetual fallback license

## Activating a License

### Online Activation

1. Click the **license icon** in the sidebar
2. Enter your license key and email
3. Click **Activate**

### Offline Activation

If your machine doesn't have internet access:

1. Click **Activate Offline**
2. Copy your device ID
3. Visit the activation portal
4. Enter device ID and license key
5. Copy the activation code
6. Paste and activate

## License Details

### Device Activations

Each Pro license includes 3 device activations:
- Work laptop
- Home computer
- Extra device

Need more? Contact support.

### Updates

Your license includes 1 year of updates from purchase date. After expiration:
- Your current version continues working forever
- New updates require renewal
- Renew anytime to get latest features

### Perpetual Fallback

If you don't renew:
- Keep using the version you have
- It doesn't stop working
- No time limits

## Checking License Status

Click the license icon to see:
- License type
- Expiry date
- Days remaining
- Activation status

## Deactivating

To move your license to a new device:

1. Open license settings
2. Click **Deactivate**
3. Activate on the new device

## FAQ

### What counts as commercial use?

Using data-peek for work-related activities in a for-profit organization of 2+ people.

### Can students use it for free?

Yes! Students and educators can use data-peek for free. Contact us for a free license key.

### Is there DRM?

No. We use an honor system. There's no aggressive license checking or "you've been logged out" surprises.

### What if my company has 1 person?

Solo founders (company of one) can use the free version.

### Can I get a refund?

Yes, 30-day money-back guarantee. No questions asked.
