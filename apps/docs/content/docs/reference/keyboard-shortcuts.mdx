# Keyboard Shortcuts

data-peek is designed for keyboard-first navigation. Here are all available shortcuts.

## Query Execution

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Execute query | `Cmd+Enter` | `Ctrl+Enter` |
| Format SQL | `Cmd+Shift+F` | `Ctrl+Shift+F` |
| Save query | `Cmd+Shift+S` | `Ctrl+Shift+S` |

## Tab Management

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| New tab | `Cmd+T` | `Ctrl+T` |
| Close tab | `Cmd+W` | `Ctrl+W` |
| Switch to tab 1-9 | `Cmd+1` to `Cmd+9` | `Ctrl+1` to `Ctrl+9` |
| Next tab | `Cmd+Option+→` | `Ctrl+Alt+→` |
| Previous tab | `Cmd+Option+←` | `Ctrl+Alt+←` |

## Connection Management

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Connection picker | `Cmd+P` | `Ctrl+P` |
| Switch to connection 1-9 | `Cmd+Shift+1` to `Cmd+Shift+9` | `Ctrl+Shift+1` to `Ctrl+Shift+9` |

## View

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Toggle sidebar | `Cmd+B` | `Ctrl+B` |
| Clear results | `Cmd+L` | `Ctrl+L` |
| Toggle dev tools | `F12` | `F12` |

## Editor

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Undo | `Cmd+Z` | `Ctrl+Z` |
| Redo | `Cmd+Shift+Z` | `Ctrl+Shift+Z` |
| Cut | `Cmd+X` | `Ctrl+X` |
| Copy | `Cmd+C` | `Ctrl+C` |
| Paste | `Cmd+V` | `Ctrl+V` |
| Select all | `Cmd+A` | `Ctrl+A` |
| Find | `Cmd+F` | `Ctrl+F` |
| Trigger autocomplete | `Ctrl+Space` | `Ctrl+Space` |

## Application

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Quit | `Cmd+Q` | `Alt+F4` |
| Hide | `Cmd+H` | - |
| Minimize | `Cmd+M` | - |
| Full screen | `Cmd+Ctrl+F` | `F11` |

## Results Navigation

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Navigate FK | `Cmd+Click` | `Ctrl+Click` |
| Copy cell | Click cell | Click cell |

## Tips

- Most shortcuts work when the editor is focused
- Tab shortcuts work globally
- Connection shortcuts work globally
- Use `?` to see available shortcuts (coming soon)
