# Getting Started

Welcome to data-peek! This guide will help you get up and running in just a few minutes.

## Overview

Getting started with data-peek is simple:

1. **Download** - Get the installer for your platform
2. **Install** - Run the installer (no configuration needed)
3. **Connect** - Add your database connection
4. **Query** - Start exploring your data

## System Requirements

Before installing, make sure your system meets these requirements:

| Platform | Requirements |
|----------|-------------|
| macOS | macOS 11+ (Big Sur or later) |
| Windows | Windows 10/11 (64-bit) |
| Linux | glibc 2.31+ (Ubuntu 20.04+, Debian 11+) |
| RAM | 4 GB minimum |
| Disk | 200 MB free space |
